# Crypto Lab

This repository contains a simple web interface for testing AES encryption, decryption, and SHA256 hashing.

## How to run

1. Install the required dependencies:
```bash
pip install -r requirements.txt
```

2. Start the web application:
```bash
python exercice_2.py
```

3. Access the web interface by navigating to the following URL in your web browser:
```
http://localhost:8080/
```
